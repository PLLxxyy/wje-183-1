# 个人所得税计算器

一个纯前端的中国个人所得税计算工具，数据保存在浏览器本地。

## 功能

- **收入计算**：支持输入月收入或年收入，实时计算应纳税额和到手收入
- **五险一金**：内置20个主要城市的社保和公积金比例，选择城市自动填入
- **专项附加扣除**：支持子女教育、房贷利息、住房租金、赡养老人、继续教育
- **详细计算过程**：从收入到起征点到各项扣除，逐步展示，支持月度/年度视角切换
- **累进税率**：完整实现7级超额累进税率（3%-45%），逐级展示计算结果
- **历史记录**：保存每次计算的输入参数和结果，支持查看和对比不同方案
- **优化建议**：分析未填满的扣除项，计算潜在节税金额

## 技术栈

- Vite + React 18 + TypeScript
- localStorage 本地存储
- 纯 CSS 样式（无第三方UI库）
- react-router-dom 路由

## 快速开始

```bash
npm install
npm run dev
```

浏览器访问 http://localhost:3000

## 项目结构

```
src/
  main.tsx            入口文件
  App.tsx             路由和布局
  pages/
    Home.tsx          计算首页（输入收入和扣除项）
    Result.tsx        计算结果页（详细过程和优化建议）
    History.tsx       历史记录页（查看和对比方案）
  utils/
    tax.ts            个税计算逻辑（累进税率7级）
    cities.ts         城市社保/公积金比例预设
```

## 注意事项

- 本工具仅供参考，实际纳税以税务部门计算为准
- 数据仅保存在浏览器本地，清除浏览器数据会丢失记录
- 五险一金比例为参考值，实际以当地政策和个人情况为准
